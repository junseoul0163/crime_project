# Raw Data

This folder contains the original, unmodified data files used in the analysis. 
Due to file size and licensing, these files may not be tracked in the repository.

## 1. FBI Crime Data (Uniform Crime Reporting)
**Source:** FBI Crime Data Explorer (CDE)  
**URL:** [https://cde.ucr.cjis.gov/](https://cde.ucr.cjis.gov/)

**How to download:**
1. Go to "Documents & Downloads".
2. Under "State Estimates", select "Estimated Crimes (1979-2024)".
3. Download the CSV file.
4. Rename it to `estimated_crimes_1979_2024.csv`.
5. Place it in this folder.

**Expected File:**
* `estimated_crimes_1979_2024.csv`

---

## 2. BLS Unemployment Data (LAUS)
**Source:** Bureau of Labor Statistics (Local Area Unemployment Statistics)  
**URL:** [https://www.bls.gov/lau/](https://www.bls.gov/lau/)

**How to download:**
1. Go to the BLS LAUS "Statewide Data" section.
2. Download the "Annual Average" tables for each year from 2010 to 2024.
3. Save each year as a separate Excel file named `laus_YYYY.xlsx`.
4. Place them in this folder.

**Expected Files:**
* `laus_2010.xlsx`
* `laus_2011.xlsx`
* ...
* `laus_2024.xlsx`
